# Руководство пользователя VM Mobile

## Назначение

VM Mobile — мобильный клиент VisualMath для просмотра лекций, участия в сессиях и выполнения заданий.

## Сценарий студента

### Вход в приложение

1. Откройте приложение
2. Введите логин
3. Введите пароль
4. Нажмите **Войти**

В текущей demo-версии подойдет любой непустой логин и пароль.

### Работа с каталогом

На экране каталога можно:

- просматривать список лекций
- искать лекции по названию, тегам и автору
- повторно открывать последнюю лекцию из кэша

### Открытие лекции

1. Нажмите **Открыть лекцию**
2. Просмотрите описание, блоки и требования
3. Нажмите **Перейти к сессии**

### Работа с сессией

На экране сессии отображаются:

- код подключения
- статус сессии
- статус подключения
- число участников
- текущий блок

Для перехода к заданию нажмите **Открыть задание**.

### Выполнение задания

В зависимости от типа задания пользователь:

- выбирает один вариант ответа
- либо вводит короткий ответ

Также отображается таймер. Если время закончится, ответ отправится автоматически.

### Результат

После отправки показываются:

- правильность ответа
- полученные баллы
- правильный ответ
- пояснение
- время выполнения

### Профиль и настройки

В профиле доступны:

- данные пользователя
- переключение темы
- включение и отключение уведомлений
- выход из аккаунта

## Оффлайн-режим

В текущей версии поддерживается локальный кэш:

- каталога лекций
- последней открытой лекции
- темы
- настройки уведомлений

Если сеть недоступна, приложение может показать сохраненный каталог.

## Ограничения текущей версии

- нет реального подключения к серверу
- нет защищенного хранения токена
- teacher flow пока не реализован
- контент лекций пока загружается из mock-данных
